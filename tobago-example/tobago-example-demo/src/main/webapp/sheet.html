Das Sheet-Control bietet eine einfache Möglichkeit tabellarische Daten darzustellen. Die wichtigsten Eigenschaften sind:
  • Paging: Es wird nur ein Teil der Gesamtliste angezeigt. Über die Pfeilknöpfe kann in der Liste navigiert werden.
  • Sortierung: Durch Drücken auf die Spaltenüberschrift kann die entsprechende Spalte sortiert werden.
  • Inline-Editing: Es können bei Bedarf Werte direkt in die Tabelle eingegeben werden.
  • Die Spaltenbreite kann direkt mit der Maus manipuliert werden.
In diesem Beispiel ist in applikationsspezifischer Sorter hinterlegt, der die Sonne immer oben erscheinen lässt.
