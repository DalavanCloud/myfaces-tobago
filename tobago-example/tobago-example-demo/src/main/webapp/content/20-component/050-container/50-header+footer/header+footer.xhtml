A header / footer can be set via the <tc:header/> respectively <tc:footer/> tag.

In this simple example, a header and a footer are set. By default the value of the fixed attribute is 'false'. The content of the header, footer and the content between are wrapped in <tc:box/> tags. The content boxes don't have a label, but a border.
