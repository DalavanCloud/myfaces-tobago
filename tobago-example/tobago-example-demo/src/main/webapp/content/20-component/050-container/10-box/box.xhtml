A content box is created with <tc:box/>.

The title of the box is set by the label attribute.

<tc:box label="Box">Content</tc:box>
Content Contentbox without a title.

A contentbox can be collapsed via collapsed attribute.

<tc:box label="Collapsed" collapsed="true">Content</tc:box>
Content Content

It's possible to add a toobar to the header of a contentbox. For that, the contentbox must contain the tag <f:facet name="toolBar">. The facet contain the <tc:toolBar> tag.
