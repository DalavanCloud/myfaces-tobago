<tc:bar/> is a container which display elements in one single line as long as there is enough space. If space is too narrow, the bar collapse. Hidden items are accessible by clicking on the bar-icon. This makes <tc:bar/> great for navigation menus.

Additionally the facet <f:facet name="brand"/> can be added. It show the 'brand', a logo a text or both on the left site of the bar.

This example show a Bar with a 'brand' facet, a menu and right sided content inside an 'after' facet.

<tc:bar>
  <f:facet name="brand">
    <tc:link label="BRAND" link="/"/>
  </f:facet>
  <tc:links>
    <tc:link label="Link One"/>
    <tc:link label="Link Two"/>
  </tc:links>
  <f:facet name="after">
    <tc:flexLayout columns="*; auto">
      <tc:in placeholder="Search" labelLayout="skip"/>
      <tc:button label="Go"/>
    </tc:flexLayout>
  </f:facet>
</tc:bar>

This example shows how to center content.

<tc:bar>
  <f:facet name="brand">
    <tc:link label="BRAND" link="/"/>
  </f:facet>
  <div class="ml-auto">
    // centered content
  </div>
  <f:facet name="after">
    // right sided content
  </f:facet>
</tc:bar>
