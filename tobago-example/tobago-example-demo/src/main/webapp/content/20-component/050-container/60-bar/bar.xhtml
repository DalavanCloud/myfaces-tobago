<tc:bar/> is a container which display elements in one single line as long as there is enough space. If space is too narrow, the bar collapse. Hidden items are accessible by clicking on the bar-icon. This makes <tc:bar/> great for navigation menus.

Additionally the facet <f:facet name="brand"/> can be added. It show the 'brand', a logo a text or both on the left site of the bar.

This example show a Bar with a 'brand' facet and several components. Buttons and links don't have any functionality.
