A <tc:suggest/> tag can be added to an inputfield to make suggestions based on the already entered text.

Tag Library Documentation: |

Select a language, after you type a character in the textfield, a list of suggestions is shown.

<tc:in label="Language" value="\#{suggestController.suggestInput}">
    <tc:suggest suggestMethod="\#{suggestController.getInputSuggestItems}"/>
</tc:in>

This example differs in a few points from the one in the 'Basics' section. There are only two suggestions shown to the user, because the maximumItems attribute is set to '2'. Also you need to type two characters before suggestions are made, because of the minimumCharacters attribute which has the value '2'. And by setting the delay to '2000', the user have to wait two seconds, before the suggestionlist is show.

<tc:suggest suggestMethod="\#{suggestController.getInputSuggestItems}"
  maximumItems="2" minimumCharacters="2" delay="2000"/>

For optimization, the filtering can be done by the client. For that, update must be 'false' and minimumCharacters have to be '0'. The filter attribute has only an effect if client sided filtering is activated.

<tc:suggest suggestMethod="\#{suggestController.getCountryLanguageList}"
  update="false" minimumCharacters="0" filter="prefix"/>
