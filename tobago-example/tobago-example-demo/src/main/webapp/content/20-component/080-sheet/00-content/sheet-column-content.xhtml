This example, demonstrate the flexiblity of the <tc:column/> tag.

Tag Library Documentation: |

The second column contain a tab group. Also a tag is used to set the maximal height of the table. As a result, a vertical scrollbar is shown. The header is hidden by set showHeader to 'false'.

<tc:sheet value="\#{sheetController.solarList}"
    var="solarobject" rows="4" showHeader="false">
  <tc:style maxHeight="500px"/>
  ...
  <tc:column label="Orbit & Period">
    <tc:tabGroup>
      ...
    </tc:tabGroup>
  </tc:column>
  ...
