A popup dialog is created with <tc:popup/>. It can be filled with components like other container.

There are basically two ways to open a popup dialog. First, put the <tc:popup/> tag within an opening component, like a button or a link. The second one is to use the <tc:popupReference/> tag.

In this example, the popup should be within a button. To make this happen, the facet-tag is also needed. In detail:
The <tc:button label="Open Popup 1"> contain <f:facet name="popup">, which contain <tc:popup id="popup1">.

This is a popup dialog.

The link tag contain a <tc:popupReference for="popup1"/>. The value of the for attribute has to be the ID of a popup dialog.

A popup can be closed with or without submitting the entered values.

To create a 'OK'-button, insert <tc:attribute name="popupClose" value="afterSubmit"/> into the button tag.

To create a 'Cancel'-button, insert <tc:attribute name="popupClose" value="immediate"/> into the button tag.
