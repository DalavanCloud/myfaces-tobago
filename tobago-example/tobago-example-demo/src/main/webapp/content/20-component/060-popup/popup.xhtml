A popup dialog is created with <tc:popup/>. It can be filled with components like other container. To open the popup dialog, use <tc:popupReference/>.

Tag Library Documentation: | | |

Press the button or click the link to open a popup dialog. Both button and link contain a <tc:popupReference for="popup1"/> tag. The value of the for attribute has to be the ID of a popup dialog.

This is a popup dialog.

A popup can be closed with or without submitting the entered values.

To create a 'OK'-button, insert <tc:attribute name="popupClose" value="afterSubmit"/> into the button tag.

To create a 'Cancel'-button, insert <tc:attribute name="popupClose" value="immediate"/> into the button tag.
