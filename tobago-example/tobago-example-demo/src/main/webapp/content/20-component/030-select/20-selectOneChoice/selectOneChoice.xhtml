<tc:selectOneChoice/> create a dropdownbox. Static entries could be created via <tc:selectItem/>. For a list from a controller, use <tc:selectItems value="\#{controller.list}"/>.

Dropdown Box with hardcoded items. The second item is disabled.

<tc:selectOneChoice label="Dropdown Box">
  <tc:selectItem itemLabel="Item One" itemValue="one"/>
  <tc:selectItem itemLabel="Item Two" itemValue="two" itemDisabled="true"/>
</tc:selectOneChoice>

Select a person. The label of the item is shown in the dropdownbox. After submitting, the value of the item is shown in the outputfield.

<tc:selectOneChoice label="Person" value="\#{selectOneChoiceController.person}">
  <tc:selectItem itemLabel="Alice" itemValue="Alice Anderson"/>
  <tc:selectItem itemLabel="Bob" itemValue="Bob Brunch"/>
</tc:selectOneChoice>
<tc:out label="Selected Person" value="\#{selectOneChoiceController.person}"/>
<tc:button label="Submit"/>

Select a planet and a moon. The list of moons depends on the selected planet.

<tc:selectOneChoice label="Planet" value="\#{selectOneChoiceController.planet}">
  <f:selectItems value="\#{selectOneChoiceController.planets}"/>
  <f:ajax render="moonbox"/>
</tc:selectOneChoice>
<tc:selectOneChoice id="moonbox" label="Moon">
  <f:selectItems value="\#{selectOneChoiceController.moons}"/>
</tc:selectOneChoice>
