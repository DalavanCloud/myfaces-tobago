The <tc:object/> tag create an iframe. External objects can be included, when Content Security Policy is activated. You may add a frame-src policy to the 'tobago-config.xml'.

<tc:object src="https://www.google.com/maps/..."/>

A <tc:object/> tag can contain a style tag. In this case it's <tc:style width="100%"/>.

There are two <tc:toolBarCommand/> within a <tc:toolBar/> in this example. The first show the island Tobago, the second show the worldmap in <tc:object id="map">. The toolbarcommands send the parameter (maps-position, maps-zoom, maps-target) to an object.js, which is shown in the objects.js-box. The JavaScript build the URL for the iframe.

<tc:toolBarCommand label="Show Tobago" omit="true">
  <tc:dataAttribute name="maps-position" value="11.249123,-60.687103"/>
  <tc:dataAttribute name="maps-zoom" value="12"/>
  <tc:dataAttribute name="maps-target" value="page:map"/>
</tc:toolBarCommand>
