Links/Commands can be grouped with the <tc:commands/> tag. Just add some <tc:link/> or <tc:commands/> tags.

Currently there is no difference between <tc:link/> and <tc:commands/>.

Tag Library Documentation: |

A simple example with three links.

<tc:commands>
  <tc:link label="Left"/>
  <tc:link label="Center"/>
  <tc:link label="Right"/>
</tc:commands>

A dropdown menu can also be created.

<tc:commands>
  <tc:link label="Dropdown" omit="true">
    <tc:link label="Action 1"/>
    ...
  </tc:link>
  <tc:link label="Center"/>
  <tc:link label="Right"/>
</tc:commands>
