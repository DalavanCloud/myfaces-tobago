To load up files to the server for JSF 2.0 and 2.1 you will need to add an multipart-config entry to the FacesServlet in the web.xml file. Since JSF 2.2 this is not required.

Here you can configure some more general information. 


      <multipart-config>
        <location>/tmp</location>
        <max-file-size>20848820</max-file-size>
        <max-request-size>418018841</max-request-size>
        <file-size-threshold>1048576</file-size-threshold>
      </multipart-config>
The first example loads up an arbitrary file. The second example contains a <tc:validateFileItem> tag with restrict the content type.
