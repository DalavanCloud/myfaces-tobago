The <tc:treeMenu/> tag can be used for menus like the one on the left side of this site. The nodes have to be <tc:treeCommand/> tags, otherwise <tc:treeMenu/> behaves the same way as <tc:tree/>. <tc:treeMenu/> set indents automatically for <tc:treeCommand/>. So the <tc:treeIndent/> tag is not needed.

In this example, a click on a node has no effect.

<tc:treeMenu value="\#{treeController.sample}" ...>
